# 📌 Relação com o Projeto Anterior

Este projeto está relacionado aos estudos anteriores sobre sustentabilidade e impactos ambientais causados pelo descarte incorreto de resíduos. O foco atual é o lixo eletrônico nas cidades de Taió, Ascurra, Rodeio e Ibirama, buscando entender como ocorre o descarte e quais soluções podem ser aplicadas para melhorar esse processo.

---

# ❓ Problema e Pergunta Guia

O descarte incorreto de lixo eletrônico pode causar sérios danos ao meio ambiente e à saúde humana, devido aos materiais tóxicos presentes em aparelhos eletrônicos.

### Pergunta Guia:
> Como melhorar o descarte correto do lixo eletrônico nas cidades de Taió, Ascurra, Rodeio e Ibirama em Santa Catarina?

---

# 🎯 Objetivo e Usuários Possíveis

## Objetivo
Criar um projeto informativo que ajude a conscientizar a população sobre a importância do descarte correto do lixo eletrônico e apresentar locais e maneiras adequadas para realizar esse descarte.

## Usuários Possíveis
- Moradores das cidades pesquisadas  
- Estudantes  
- Professores  
- Empresas de reciclagem  
- Órgãos públicos municipais  

---

# ⚙️ Funcionalidades Propostas

- Informações sobre os riscos do descarte incorreto  
- Divulgação de pontos de coleta nas cidades pesquisadas  
- Conscientização ambiental  
- Explicação sobre reciclagem de eletrônicos  
- Organização de dados em planilhas  
- Sistema simples para consulta de locais de descarte
